# SecureAzCloud Post-Quantum Migration Hub

## Overview

The SecureAzCloud Post-Quantum Migration Hub is a public resource center dedicated to helping organizations prepare for the transition from quantum-vulnerable cryptography to NIST-standardized Post-Quantum Cryptography (PQC).

This repository provides practical implementation guidance, assessment tools, migration planning resources, and operational checklists designed for enterprise, cloud, identity, and critical infrastructure environments.

The content is vendor-neutral and aligned with publicly available standards and guidance from NIST, CISA, and related cybersecurity frameworks.

## Included Resources

### Cryptographic Inventory Template

A practical workbook for identifying and cataloging:

- Cryptographic assets
- Algorithms and key sizes
- Certificates
- Identity systems
- PKI dependencies
- Cloud workloads
- Third-party services
- Operational technology environments

### PQC Readiness Assessment Checklist

A structured assessment framework to help organizations evaluate:

- Governance readiness
- Asset visibility
- Crypto-agility maturity
- Identity modernization readiness
- Vendor preparedness
- Operational risk

### Cloud and IAM Migration Playbook

Guidance for planning migration activities involving:

- Identity platforms
- PKI modernization
- Certificate lifecycle management
- Workload identities
- Machine authentication
- Hybrid cloud environments

### SCADA and ICS Readiness Checklist

Operational technology-focused guidance covering:

- Industrial control systems
- SCADA environments
- Safety considerations
- Vendor dependencies
- Legacy protocol assessments
- Migration planning activities

## Standards Alignment

This resource hub aligns with publicly available guidance including:

- NIST FIPS 203 (ML-KEM)
- NIST FIPS 204 (ML-DSA)
- NIST FIPS 205 (SLH-DSA)
- NIST Cybersecurity Framework 2.0
- NIST NCCoE Migration to PQC Project
- NIST SP 800-82 Rev. 3
- NIST SP 800-161 Rev. 1 Update 1
- NIST Crypto Agility Guidance (CSWP 39)
- CISA Quantum Readiness Guidance
