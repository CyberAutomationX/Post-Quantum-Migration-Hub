# PQC Migration Resource Hub

The SecureAzCloud PQC Migration Resource Hub is an open and vendor-neutral collection of practical resources designed to help organizations prepare for the transition from quantum-vulnerable cryptography to NIST-standardized Post-Quantum Cryptography (PQC).

The repository provides implementation-focused guidance for cryptographic discovery, inventory management, crypto-agility planning, cloud and identity modernization, and operational technology (OT), industrial control systems (ICS), and critical infrastructure environments.

Resources align with NIST PQC standards, the NIST Cybersecurity Framework 2.0, and industry-recognized cryptographic migration practices.
