# SecureAzCloud PQC Migration Resource Hub

Version: 1.0  
Release date: 2026-06-07

This package contains a vendor-neutral, NIST-aligned resource hub for post-quantum cryptography migration preparation.

## Contents

- `index.html` — hub landing page.
- `downloads/secureazcloud-pqc-crypto-inventory-template.xlsx` — workbook with crypto inventory, scoring, readiness checklist, cloud/IAM playbook, OT/ICS checklist, and references.
- `downloads/secureazcloud-pqc-crypto-inventory-template.csv` — lightweight crypto inventory import template.
- `resources/pqc-readiness-checklist.html` — public readiness checklist.
- `resources/cloud-iam-migration-playbook.html` — cloud/IAM migration playbook.
- `resources/scada-ics-pqc-checklist.html` — SCADA/ICS readiness checklist.
- `resources/reference-map.html` — source and framework reference map.
- `publishing-guide.md` — launch instructions.

## Recommended use

1. Publish the static site folder on a public website or internal portal.
2. Use the workbook to collect cryptographic inventory and risk scoring.
3. Use the readiness checklist to organize program governance and evidence.
4. Use the cloud/IAM playbook to plan identity, PKI, certificate, token-signing, and workload-identity migration.
5. Use the SCADA/ICS checklist for safety-first operational technology planning.

## Source alignment

This hub uses public NIST and CISA sources, including NIST FIPS 203, FIPS 204, FIPS 205, NIST NCCoE Migration to PQC, NIST CSWP 39 on crypto agility, NIST SP 800-82 Rev. 3, NIST SP 800-161 Rev. 1 Update 1, NIST SP 800-53 Rev. 5, and CISA PQC migration/OT resources.

This package is an independent planning aid and does not imply endorsement by any public agency.
