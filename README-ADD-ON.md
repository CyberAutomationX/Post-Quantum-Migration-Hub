# SecureAzCloud PQC Add-on Implementation Artifacts v1.1.0

This package adds three implementation-focused, public/redacted artifacts to the existing SecureAzCloud PQC Migration Resource Hub.

Existing live hub:

https://cyberautomationx.github.io/Post-Quantum-Migration-Hub/

## What is new in this add-on

1. **Crypto-Agility Assessment Template**
   - PDF/XLSX/CSV
   - Crypto asset inventory fields
   - Long-lived data flag
   - Risk scoring
   - P1/P2/P3 migration priority
   - Supplier-readiness and evidence fields

2. **PQC Cloud & Workload Identity Migration Checklist**
   - PDF/XLSX/CSV
   - Certificates, workload identities, PKI, TLS/mTLS, token signing, key lifecycle, SaaS dependencies
   - Mapped to CISA 2026 PQC product-category guidance for cloud services, web software, and endpoint security

3. **SCADA/ICS PQC Migration Continuity Checklist**
   - PDF/XLSX/CSV
   - Latency, legacy gateway, maintenance-window, vendor-dependency, hybrid-mode, rollback, and safety-continuity controls

## Public/redacted design

These artifacts do not contain secrets, real hostnames, IP addresses, customer names, internal diagrams, proprietary configurations, or key material. Sample rows are examples only.

## Source alignment

The artifacts are aligned to NIST FIPS 203, FIPS 204, FIPS 205, NIST NCCoE Migration to PQC, NIST CSWP 39, NIST SP 800-82 Rev. 3, NIST SP 800-161 Rev. 1 Update 1, and CISA PQC adoption resources.
